Local Storage Demo

Create a record:

  • var d = new Data();
  • var fname = "Joe", lname = "Schmoe";
  • var r = d.createRecord(fname, lname);
  • This creates a new record, increments sequence and hashes it to create the id, and writes the record to localStorage
  • If the new record is the first one, this will also create 'sequence' and 'recordIndex' in localStorage
  • r.id: see the id of the newly created record
  • r.fname: see first name
  • r.lname: see last name

Load a record with id:

  • var d = new Data();
  • var r = d.readRecord(id);
  • This retrieves the record from localStorage

Update a record:

  • var d = new Data();
  • var r = d.updateRecord(id, fname_new, lname_new);
  • This first loads the record from localStorage, then writes the new data

Delete a record:

  • var d = new Data();
  • var r = d.deleteRecord(id);
  • This deletes the record from localStorage and returns the deleted record

Load all records:

  • var d = new Data();
  • d.readAll();

Delete all records:

  • var d = new Data();
  • d.clear();
